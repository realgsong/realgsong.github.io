1024

65536

65536

Obviously, (f n) = 2n

(g n) = (A 1 n)
=>
(A 0 (A 0 (A 0 ...(A 0 1)))...)

이 된다. (A 0 n) = 2n 이므로 위 식은 결국
2 * 2 * 2 * ... * 2 = 2^n

(h n) = (A 2 n)
=>
(A 1 (A 1 (A 1 ...(A 1 1)))...)

이 된다. (A 1 n) = 2^n 이므로 이 식은 2^(2^(2^(...2))) 이 된다. 수학식으로 쓰자면

A(1) = 2
A(n) = 2^A(n-1)

라 할 때 (h n) = A(n) 이 된다.
