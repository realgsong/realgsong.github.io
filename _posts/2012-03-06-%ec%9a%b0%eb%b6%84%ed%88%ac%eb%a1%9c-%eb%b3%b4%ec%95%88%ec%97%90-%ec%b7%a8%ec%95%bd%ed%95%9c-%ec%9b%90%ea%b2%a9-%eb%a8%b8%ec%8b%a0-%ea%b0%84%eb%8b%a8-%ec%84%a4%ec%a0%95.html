보통 원격 로그인으로 작업하는 PC 는 모니터를 잘 연결하지 않는다. 이런걸 Headless machine 라고 부르는 모양인데, 우분투에서 이렇게 사용하려면 문제가 좀 있다.

문제의 원인은 VNC server 가 로그인을 해야만 작동한다는 것이다. 그래서 검색을 좀 해보면, 로그인 하기 전에 VNC Server 를 띄우는 여러가지 방법들이 나오는데, 하나같이 설정하기 귀찮다.

그러다 얻어 걸린 보안에 취약한 방법 하나를 옮겨온다.

  1. Desktop Sharing 에서 원격 연결을 허용토록 하고, 적절한 패스워드를 설정한다.
  2. Screen 에 가서 30 초 후에 Lock Screen 이 되도록 한다.
  3. User Accounts 에서 해당 사용자 계정에 automatic login 을 설정한다.
