CODE 코드 -
찰스 펫졸드 지음, 김현규 옮김/인사이트

찰스펫졸드가 지은 책이다. 구상은 1987년 부터 시작했다고 하고, 근 10년간 머리 속을 떠돌다가  초판은 96년에서야 나왔다. 우리말 번역본은 2010 년에 나왔으니 20년도 더 된 고전이다.

컴퓨터 공학을 수직으로 관통하는 내용으로, 모스 부호부터 키보드구현을 통한 입력까지 컴퓨터의 모든 걸 다룬다. 이 책을 다 읽고 나면 컴퓨터라는 기계에 전원이 공급될 때 어떤 일들이 일어나는지 이해할 수 있을 것 같은 자신감이 생길 것이다.

그리고 무엇보다 번역이 매끄럽게 참 잘되었다. 읽는 데 막힘이 없고 내용의 전달에도 충실하다. 간간히 더해져 있는 역자주 는 책 이해를 돕는데 충분하다.

회로 이론, 논리 설계, 프로그래밍 언어, 운영체제 등등의 컴퓨터 공학 과목들을 배우게 되는데, 이것을 하나로 꿰어 내기는 쉽지 않다. 펫졸드의 이 책을 보고 나면 왜 소프트웨어를 배우기 위해 하드웨어를 익혀야 하는지 이해하게 될 것이다.
