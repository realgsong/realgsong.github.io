아래 내용은 최신 버전 에는 맞지 않습니다. Ubuntu 12.10 이상에서 여기 에서 developer tool 받아다 압축 풀고 실행하시면 문제 없이 이클립스가 뜹니다.

Ubuntu 12.04 64bit 버전에 안드로이드 개발 환경 갖추기

cd ~/
wget https://raw.github.com/flexiondotorg/oab-java6/master/oab-java.sh -O oab-java.sh
chmod +x oab-java.sh
sudo ./oab-java.sh
sudo apt-get install sun-java6-jdk
  1. JDK6 설치하기가 조금 까다롭다. 다행히 용자분이 계셔서이미 해결해두었음. 위 스크립트 실행
  2. 여기에서 eclipse 다운로드 받아 적당한 위치에 압축 풀어 두기.
  3. 여기에서 ADT Plugin 다운로드.
  4. eclipse 실행한 다음  Help < Install New Software 에서 Add 누르고 위에서 다운로드한 zip 파일 선택
  5. sudo apt-get ia32-libs
  6. 나머지는 ADT plugin 을 통해 모두 설치. ia32-libs 없으니까 adb 명령이 안돌아감.
