Normal order 를 가정했을 때
(test 0 (p))
(if (= 0 0) 0 (p)))
0 가 리턴되고 끝
 
Applicative order 를 가정했을 때
(test 0 (p))
evaluate the arguments
(test 0 (p))
evaluate 작업을 무한히 반복하게 됨.
