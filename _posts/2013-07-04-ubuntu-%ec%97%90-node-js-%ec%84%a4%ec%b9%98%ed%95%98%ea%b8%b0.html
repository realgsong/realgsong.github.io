Node.js 우분투 셋업

node.js 버전업이 빨라서 그런지 Ubuntu packaging 에 들어가 있는 건 버전이 낮나보다. node 라는 이름도 다른 프로그램이 이미 선점한 듯. 다행히 PPA 로 최신 노드를 설치할 수 있게 해주신 님이 있어서 그걸 이용하면 됨.

sudo apt-get install python-software-properties python g++ make
sudo add-apt-repository ppa:chris-lea/node.js
sudo apt-get update
sudo apt-get install nodejs npm
