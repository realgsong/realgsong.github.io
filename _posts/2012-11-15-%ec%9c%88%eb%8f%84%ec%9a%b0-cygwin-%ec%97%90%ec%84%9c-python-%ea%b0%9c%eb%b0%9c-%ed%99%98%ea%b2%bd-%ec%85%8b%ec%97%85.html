Anything simple: Using pip, virtualenv, and virtualenvwrapper in Cygwin

pip 설치

Python 인터프리터와 PIP 그리고 Virtualenv 를 설정하여 파이썬 개발 가능한 환경을 Cygwin 에서 구축하는 방법에 대한 것이다.

  1. 먼저 여기에서 cygwin 다운로드 받고 실행.
  2. 설치할 component 목록에서 Python 선택 후 설치. wget 유틸도 함께 설치. 필요하다면 vim 같은 에디터도 설치한다.
  3. easy_install 설치. http://peak.telecommunity.com/dist/ez_setup.py 를 다운로드해서 실행한다.
  4. easy_install pip
  5. pip install virtualenv
  6. pip install virtualenvwrapper
  7. $HOME/.bash_profile 에 다음 줄 추가. WORKON_HOME 의 값은 적당한 디렉토리로 설정. 
    export WORKON_HOME=/cygdrive/c/Users/userid/dev/virtualenvs
source /usr/bin/virtualenvwrapper.sh
